Searching

Enter the Twitter/Instagram username of the person you are trying to find. The search will contain the characters as a prefix of whatever you enter.

Hint: The search will autosearch as you type but will only show up to 20 usernames on the screen at once.

Search Results

Understanding Search Results

Newman returns your search results to a column showing up to 20 total users. The users will have an icon next to them which represents what social media platform the user belongs to. Selecting the username will take you to our profile analysis page for that person.

Viewing a Profile

At the top of the screen will display the user selected and social media icon associated to the username. Selecting the search user will allow you to go back to the search page.

The left side panel shows the visual representation of the user's posts and associations with other users while the right side panel shows information such as the associated users, possible aliases, hashtags, and the content of any selected post.

At the very bottom of the screen is a table list of all of the selected users posts. This can be collapsed and expanded to give more viewing room to the graph/information panels.

Shared Attribute Graph

Graph Display

The graph displays all of the user's posts as nodes and the users associated to each post. These associations are made through actions such as mentions, likes, retweets, comments, etc.

Graph Interaction

Pan The Graph

  • Left-click on an empty part of the graph space and hold down the mouse button Then, drag the mouse while holding down the mouse button to move the graph.

Selection

  • Left-click on a user node and select the magnifying glass to show that user's graph display.
  • Left-click on a post node and select the magnifying glass to show the content of the post in the information panel.

Zooming

  • While your mouse is over the graph, scroll up to zoom in
  • While your mouse is over the graph, scroll down to zoom out

 The legend can be used to filter out the node labels. Just check/uncheck whatever you want to see.

Information Panel

The panel on the right of the screen has four panes: Associated Users, Possible Aliases, Hashtags, and Content.

Associated Users

The Associated Users pane shows the top 20 users associated with the current selected user. The bars represent how often that user associates with the user. These associations are quantified through actions such as mentions, likes, retweets, comments, etc.

Possible Aliases

The Possible Aliases pane shows all of the possible aliases for the selected user based on the MIT rankings which uses a custom algorithm to determine what aliases this user may be using. The bars represent the MIT score.

Hashtags

The Hashtags pane shows the top 20 hashtags the selected user uses in their posts. The bars represent the quantity the user has used the hashtag in their posts.

Content

The Content panel displays information on the selected post:

  • Post - The actual text of the post made by the user.
  • URL - The link to the actual post made by the user.
  • Associated Users - Lists the users that associated with this post.
  • Tags - List of hashtags that were used in this post.
  • User Post - The embedded version of the actual user post.
